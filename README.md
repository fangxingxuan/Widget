# Widget
fangxingxuan android common widget library

# Feature
- pie chart view


# How to use

Step 1. Add the JitPack repository to your build file
```
repositories {
    ...
    maven { url 'https://jitpack.io' }
}
```


Step 2. Add the dependency
```
dependencies {
    compile 'com.github.fangxingxuan:Widget:1.0.0'
}
```
